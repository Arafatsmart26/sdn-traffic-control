# SDN Traffic Control

## Overview

This project explores Software-Defined Networking (SDN) by implementing traffic control and network management using tools like Mininet and the Ryu controller.

## Status

🚧 Project in early development — setup and initial implementation coming soon.

## Objectives

* Understand the fundamentals of SDN architecture
* Build a simulated network using Mininet
* Implement a controller to manage and control traffic flows
* Analyze network behavior under different conditions

## Tools & Technologies

* Python
* Mininet
* Ryu Controller

## Planned Features

* Custom network topology
* Traffic monitoring
* Flow control and prioritization
* Basic performance analysis

## Notes

This project is being developed independently during my free time as a self-learning initiative.

AI-based tools are used as supplementary aids for understanding concepts and guidance, while implementation is done independently.
